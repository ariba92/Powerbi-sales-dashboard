# Sales Performance Dashboard

## Overview
An interactive Power BI dashboard developed to analyze sales performance across different countries, products, and customer segments.

## Tools Used
- Power BI
- Excel
- DAX
- Data Cleaning

## Features
- KPI Cards
- Sales by Month
- Profit by Product
- Profit by Country
- Profit by Segment
- Interactive Filters
